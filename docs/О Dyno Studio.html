Dyno Studio - это

Создание пользовательского интерфейса для скриптов Dynamo

Скачайте последнюю или старые версии

А если подробнее

Dyno Studio - это среда, которая позволит сделать ваши скрипты по-настоящему профессиональными и законченными

Проект состоит из нескольких этапов, каждый из которых реализует дополнительный функционал. Этот функционал будет внедряться в Dyno Studio постепенно

Stage 1

Создавайте формы и пользовательский интерфейс для скриптов с помощью простого и понятного редактора

Stage 2

Упаковывайте свои скрипты и все зависимости в инсталляторы. Делитесь с коллегами, осуществляйте распространение и устанавливайте свои инструменты в один клик

Текущий этап: Stage 1

5 вещей, которые поднимут работу с Dyno и Dynamo на новый уровень

Создавайте интерактивный пользовательский интерфейс для своих Dynamo инструментов, делая их более доступными и профессиональными!

1

Создавайте множество форм для одного скрипта

Конструктор пользовательского интерфейса сделан простым и понятным 

Любой проектировщик или BIM специалист сможет потратить немного времени и освоить создание своих форм.

Пользовательский интерфейс может выполнять разные функции

Каждый скрипт может иметь главное окно (по аналогии с главным окном программ) и диалоговые окна, которые появляются при определенных обстоятельствах в процессе работы скрипта.

Более того - каждое главное окно или диалоговое окно может иметь несколько вкладок для более удобного и оптимального расположения элементов управления.

1.1 Program settings and UI form creating
1.2 Showing the form as main script window
1.3 Showing the form as dialog window in the script

2

Конструируйте формы используя универсальные компоненты

Dyno Studio содержит базовые универсальные компоненты для форм

Такие компоненты, как "Кнопка", "Текстовое поле", "Текстовая метка", "Флажок" и другие позволят реализовать любой функционал.

В дальнейшем количество элементов управления будет увеличиваться!

2.1 Placing controls on form and manipulating them

3

Связывайте элементы формы с данными

Используйте разные источники данных для элементов форм

Источниками данных могут служить входные ноды, параметры внутри пресетов Dyno или любые данные, которые генерируются в процессе работы скрипта.

Элементы управления умеют работать с любыми данными Dynamo.

При изменении данных в форме с помощью элементов управления, они уходят далее в работу и используются скриптом.

3.1 Bindings from test Parameters
3.2 Bindings from nodes
3.3 Bindings from node port data in dialogs

4

Делайте формы интерактивными

Вы можете не только отображать и изменять данные с помощью элементов управления

Элементы управления могут использовать данные и специальные логические выражения, чтобы менять свое поведение и графику.

Это позволяет делать пользовательский интерфейс интерактивным и меняющимся в зависимости от настроек или данных.

3.6 Expression bindings and interactive UI

5

Проверяйте вводимые пользователем данные

Текстовые поля и сама форма могут проверять введенные пользователем данные

Используя систему регулярных выражений вы можете очень гибко контролировать вводимые данные. Например ограничить ввод только целыми числами, или текстом, или десятичными числами с двойной точностью и т.д.

При вводе недопустимых значений элемент управления будет выделен. Также форма на которой есть элементы с неправильными данными не даст запустить скрипт.

Вопросы и ответы про лицензии

Это платная программа?

Да. Для возможности сохранения созданных интерфейсов вам необходимо приобрести одну из предлагаемых лицензий.

А есть бесплатная демонстрационная версия?

Вы можете скачать версию программы совершенно свободно. Бесплатный режим полностью функционален за исключением возможности сохранения созданного пользовательского интерфейса. Этот режим идеален для изучения программы и знакомства с её функциями на прилагаемых примерах.

Платная версия отличается только возможностью сохранения?

Да, Полный режим позволяет также сохранять созданный пользовательский интерфейс для дальнейшего использования с Dyno. Для включения полного режима вам необходимо приобрести одну из предлагаемых лицензий.

Лицензии привязываются к оборудованию компьютера?

Да.

Могу ли я перенести лицензию с одного компьютера на другой?

Да. Но количество возможных переносов (реактиваций) зависит от типа лицензии. Лицензии, расчитанные на использование на разных устройствах и частый перенос позволяют делать много реактиваций.

Вам не нужно покупать лицензию Dyno Studio для использования уже созданных пользовательских интерфейсов в Dyno. Dyno работает с пользовательскими интерфейсами независимо от Dyno Studio!
